# 使用 Python 的生成式人工智能

在这个代码库中，你可以找到一些关于生成式人工智能 (GenAI) 和大语言模型 (LLM) 应用的实用示例。  
所有项目均在 [AI悦创](https://bornforthis.cn/) 上有详细文档。
